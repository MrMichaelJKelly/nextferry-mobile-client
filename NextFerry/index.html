About

The NextFerry app was created by Denise Draper, because I often want to know whether I should take the Edmonds or the Bainbridge ferry. Email denisesandbox@mailup.net for information, feedback or requests.

The NextFerry app is not affiliated with WSDOT. The NextFerry schedule is updated every season when the WSDOT schedule changes, however delayed updates or errors are possible. Also, NextFerry uses a simplified weekday/weekend schedule, which misses some irregular departures on some routes (typically Friday-only or Saturday-only). Consult the official WSDOT schedule or website (http://wsdot.gov/ferries) for definitive information.

This version of the NextFerry app is an HTML5 cross-platform mobile application that communicates with a backend service hosted on the Google App Engine. The code for app and server can be found at https://github.com/denised/nextferry. The client uses jQuery, iScroll and the Dosis font; liscencing info for each can be found in the github source. Travel times are provided by MapQuest.
